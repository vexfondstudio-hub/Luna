# LunaVoice Mobile — войсчат для Minecraft Bedrock прямо с телефона (Termux, без ПК)

## Установка на телефон (Android)
1. Поставь **Termux** (F-Droid или напрямую apk — из Google Play версия устарела).
2. В Termux:
```
pkg update && pkg install nodejs -y
```
3. Перенеси папку `mc-voice-mobile` на телефон (например, распакуй архив через "Файлы" в `/sdcard/Download/`), затем:
```
termux-setup-storage
cd /sdcard/Download/mc-voice-mobile
npm install
npm start
```
Увидишь: `Открой в браузере телефона: http://localhost:19131`

4. Не закрывай Termux (сверни, но не выгружай из памяти) — можно один раз выполнить `termux-wake-lock`, чтобы Android не убил процесс.

## Подключаем мод (LunaVoice.mcpack)
1. Установи `LunaVoice.mcpack` в Minecraft (двойной тап по файлу).
2. Добавь пак в мир (Behavior Packs) и включи **Cheats: ON**.
3. Ничего вводить руками не нужно — мод сам шлёт `/connect localhost:19131` при заходе в мир, сервер уже слушает этот порт.

## Голос
1. Открой в браузере телефона (Chrome/Firefox) ровно: `http://localhost:19131` — именно `localhost`, не IP, иначе браузер не даст доступ к микрофону.
2. Разреши доступ к микрофону.
3. Держишь большую кнопку на экране — говоришь; отпустил — молчишь.
4. Сворачиваешь браузер (страница должна остаться открытой в фоне), заходишь в Minecraft — общаетесь.

## Условие для двух игроков
- Оба телефона должны быть в **одной Wi-Fi сети** (обнаружение друг друга идёт через UDP-broadcast по локальной сети — через мобильный интернет не сработает).
- На каждом телефоне отдельно: свой Termux + свой сервер + своя открытая страница + свой `LunaVoice.mcpack`.

## Если что-то не так
- Координаты не обновляются → открой на телефоне `http://localhost:19131` через ПК-браузер по тому же Wi-Fi (или проверь консоль Termux) — глянь, что реально прилетает в подписке `PlayerTravelled`/`PlayerTransform`, имя события может отличаться по версии Bedrock.
- Игроки не видят друг друга в списке → роутер может блокировать broadcast между устройствами (AP isolation) — тогда нужен другой Wi-Fi или роутер с отключённой этой опцией.
- Termux "засыпает" в фоне → `termux-wake-lock` в терминале или отключить оптимизацию батареи для Termux в настройках Android.
